Basics

HTML stands for Hyper Text Markup Language and it is an interface used on almost all webpages. This structure can be thought of like writing an essay. You have an essay split into multiple parts: an intro, a thesis statement, a body paragraph, and a conclusion. The required components of HTML come together to make an entire webpage.

The most popular structure consists of the type of HTML (Doctype) in the intro, a html that is responsible for the whole page, and a body that cover the main section of the webpage. There are many other sections that can be used to structure the webpage but these are the ones needed for every webpage.

  • <!DOCTYPE html> - tag that sets the type of html.
  • <html> - main tag that the whole webpage goes into.
  • <body> - tag that the main section of webpage goes into.

Headings

Websites have different titles to announce new sections. This is done by having a header with the topic of the section. There are six different types of headers that can be used for HTML webpages.

Headings are straight forward with their codes. They start with h1 to include the largest heading and can be any number 1 through 6, 6 being the smallest. The different headings help keep track of the sections of the page because bigger headings mean more important sections and they keep the flow of the webpage.

  • <h1>Big Text</h1> - largest heading tag
  • <h6>small text</h6> - smallest heading tag

Paragraphs

An essay can't be an essay without paragraph. Similarly a html webpage cant be an html webpage without the p element or other sections. Paragraphs are a common way of sectioning the different parts of a webpage. They are most commonly found in the body tag.

There are other ways to create sections in HTML. Other tags like section, main, header, article, div, and others can represent a specific section in your webpage. This helps web developers organize their webpages by labeling the sections with diferent tags.

  • <p> - main paragraph tag
  • <section> - tag to label a section in webpage
  • <article> - specific tag for articles and references
  • <div> - common tag for a variety of miscellaneous sections

Formatting

Webpages tend to have a lot of text in their content so it is important to have the correct formatting for it all. When creating a document, making text bold, in italics, or underlined are important to the meaning and structure. Similiarly, webpages can use the same formatting in its text to get the message across.

There are many formatting options for the text, even some that are lesser known. Examples include highlighting text, creating superscripts and subscripts, striking through text, and creating smaller text. There are many others that can be researched.

  • <b> or <strong> - Bold tag
  • <i> or <em> - Italics tag
  • <ins> - Underline tag
  • <mark> - Highlight tag

Colors

To add color to your webpages, the easiest way is to use CSS principals. CSS stands for Cascading Style Sheets and is another coding program that lets you style your elements in html. The most common two ways of adding color into your webpage is by adding a background color or changing the color of your text or element. Instead of having to change the color and styles of different paragraphs and elements in your webpage, CSS allows you to change multiple elements at a time by grouping them in classes.

There are a few different ways of listing colors as a value. The easiest way is by writing out the names such as red, blue, or white. You can even write out less popular colors such as dodgerblue or cyan. Another way is by writing the color hex code. These series of numbers and letters allow for 256 color combinations. Another way is entering in the combination of reds, greens, and blues in a color.

  • {background-color: ;} - changes the background color
  • {color: ;} - changes the text/element color
  • { : #eee555;} - example of a hex code
  • { : red;} - example of a color tag
  • { : rgb(220,40,60);} - example of a rgb code
